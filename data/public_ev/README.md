# Public EV Data

Large public EV charging-session CSV files are not stored in this repository.

For the Palo Alto ChargePoint replay, download the City of Palo Alto electric vehicle charging station usage dataset and save it as:

```text
data/public_ev/palo_alto_chargepoint.csv
```

Alternatively, set `PUBLIC_EV_DATA_PATH` to a local CSV path before running `simulations/public_ev_validation.py`.
